## 🛠️ 当前项目：Server-Sentry (全端口哨兵)
这是我为了监控服务器安全而手搓的一个轻量级 Python 脚本。

### 核心功能：
- **全端口监控**：实时扫描所有 IPv4 入站连接。
- **智能预警**：发现新连接时，自动通过钉钉机器人推送详细信息。
- **频率限制**：内置冷却机制，在遭遇高频连接时自动开启静默保护，防止服务器卡死。

### 快速开始：
1. 安装依赖：`pip install psutil requests`
2. 配置 Token：在脚本中填入你的钉钉机器人 Token。
3. 运行：`python3 sentry.py`
